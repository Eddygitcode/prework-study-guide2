HTML

    • The head element contains information about the webpage.
    • The body element represents the visible content shown to the user.
    • Content is wrapped in tags also known as elements.

CSS

  • CSS controls the style of a page.
  • The CSS is linked to the HTML in the "head" section.

Git

  • Git is used for version control.
  • You can use SSH keys to avoid logging in and share your code from your computer directly to Github.

JavaScript

  • Javascript is what makes a web page interactive.
  • Javascript can act as both a front-end and back-end code.
  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • You must call a function in order to run that piece of code.
  • You call a function by listing the function name followed by "()".
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • When debugging, it is helpful to have someone else test your code.
  • A styles sheet must be linked to the HTML in the head tag.
